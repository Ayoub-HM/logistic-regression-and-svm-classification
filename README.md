# From Linear Boundaries to Kernels — Logistic Regression & SVM (Linear + Non-Linear)

A compact Jupyter Notebook lab focused on **supervised classification** using:
- **Logistic Regression**
- **Linear SVM**
- **Non-linear SVM (kernel methods)**

This project is designed to compare how decision boundaries evolve from **linear** models to **non-linear** separators using kernels.  

## What you’ll practice
- Data exploration and preprocessing (basic)
- Training and comparing classifiers (LogReg vs SVM)
- Understanding linear vs kernel-based separation
- Visualizing decision boundaries
- Interpreting results (accuracy / confusion matrix depending on the notebook)

## Project structure
- `02-Classification/` : main notebook(s) for the classification TP
- `__MACOSX/` : leftover folder (safe to ignore)
